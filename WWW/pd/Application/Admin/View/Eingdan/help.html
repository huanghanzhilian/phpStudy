手动下单使用帮助

  • 01选择服务地址

  • 02选择服务项目

  • 03填写客户信息:客户姓名,客户电话,上门服务地址

  • 04选择客户是否已经收到配件，若尚未收到配件，须填写客户对应的淘宝订单的物流编号；若已收到配件则可直接下单

    尚未收到配件

    已经收到配件

  • 05点击立即下单按钮，下单成功

自动下单使用帮助

  • 01进入店铺管理页面

  • 02点击“开启自动下单”按钮

  • 03输入店铺授权代码并点击下方提交按钮

  • 04提交完成后请耐心等待维百士审核，审核通过后您将收到维百士的短信通知

  • 05审核通过后您可以对开启自动下单的店铺进行旗标和合作宝贝的相关设置

淘宝主图替换帮助

  • 01使用Photoshop打开维百士提供的源文件

  • 02双击图层面板中的智能对象，在新打开的窗口中替换默认图片为自家宝贝主图

  • 03把自己家的主图拖入到PS新打开的窗口中

    按回车（Enter)键确定，并按Ctrl +S 保存

  • 04回到刚打开源文件时的第一个窗口，此时默认图片已被自己家宝贝的主图替换

    Ctrl +Shift+S 另存图片为png图片

  • 05上传刚保存的图片到合作宝贝主图
